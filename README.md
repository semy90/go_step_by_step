# go_step_by_step

Добро пожаловать в репозиторий, содержащий мои решения для трёх курсов по языку программирования Go:

- **Go-1** – базовый курс (основы синтаксиса, работа с типами данных, управляющие конструкции, функции, пакеты)
- **Go-2** – продвинутый курс (конкурентность, обработка ошибок, тестирование)
- **Go-3** – углублённый курс (сетевое программирование, создание веб-сервисов, инструменты разработки)

Каждый курс разбит на модули (темы), и в соответствующей папке вы найдёте все выполненные задания.

## Структура репозитория
├── go-1/  
│ ├── level-1/  
│ ├── level-2/  
│ └── ...  
├── go-2/  
│ ├── step-1/  
│ ├── step-2/  
│ └── ...  
├── go-3/  
│ ├── step-1/  
│ ├── step-2/  
│ └── ...  
└── README.md
